# Amazon Front-End Clone

A multi-page **Amazon-style front-end clone** built using **HTML, CSS, and JavaScript**.  
This project focuses on recreating the core layout, styling, and user interface of an e-commerce platform, including product pages, authentication screens, and a shopping cart.

## Pages Included
- **Home page** (`index.html`)
- **Product details page** (`product.html`)
- **Shopping cart** (`cart.html`)
- **Sign in page** (`signin.html`)
- **Sign up page** (`signup.html`)

## Features
- Responsive navigation bar and layout
- Product listings and sliders
- Detailed product page layout
- Shopping cart UI
- Sign in and sign up form pages
